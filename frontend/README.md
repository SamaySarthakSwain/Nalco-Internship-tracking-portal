# Internship Tracking Portal

## Overview
A smart AI-powered Internship Tracking Portal that helps colleges monitor internships, improve student outcomes, automate tracking, and provide data-driven insights. Built with React (Vite), Tailwind CSS, Framer Motion, Three.js, and Shadcn UI.

## Tech Stack
- **Frontend Framework**: React + Vite
- **Styling**: Tailwind CSS + Shadcn UI
- **Animations**: Three.js (Aurora Background), Framer Motion
- **Icons**: Lucide React
- **Charts**: Recharts
- **Routing**: React Router

## Project Features Implemented
1. **Landing Page**: A futuristic and highly responsive landing page showcasing an interactive Three.js Aurora shader background for a premium feel.
2. **Authentication Flow (Login)**: Clean multi-role login portal using glassmorphism styling, covering Student, Faculty, Placement Officer, and Admin.
3. **Student Dashboard**: 
    - Full sidebar navigation setup.
    - Statistical overview cards for active applications, hours logged, and AI readiness score.
    - Task completion chart using Recharts.
    - AI recommendations widget.
